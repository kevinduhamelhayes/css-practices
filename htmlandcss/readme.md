# Galería de Elementos CSS

Este proyecto contiene una colección de elementos creados con HTML y CSS puro, demostrando diferentes técnicas de animación y diseño.

## Elementos incluidos

1. **Taza de café** - Con vapor animado que sale de la taza.
2. **Lápiz** - Con animación de escritura.
3. **Botón 3D interactivo** - Con efectos hover y click.
4. **Globo aerostático** - Con animación de vuelo y nubes.
5. **Reloj analógico animado** - Con manecillas que se mueven en tiempo real.
6. **Libro abierto** - Con efecto 3D y animación de página.
7. **Semáforo con luces animadas** - Con secuencia de luces realista.
8. **Camisa o remera** - Con pliegues y efectos de hover.
9. **Televisor antiguo** - Con efecto de estática en la pantalla.
10. **Guitarra** - Con cuerdas animadas.

## Cómo usar

Simplemente abre el archivo `index.html` en tu navegador para ver la galería completa, o abre cualquiera de los archivos HTML individuales para ver cada elemento por separado.

## Tecnologías utilizadas

- HTML5
- CSS3 (Animaciones, Transformaciones, Flexbox, Grid)
- JavaScript (solo para el reloj analógico)

## Características

- Diseño responsive
- Animaciones suaves
- Interactividad con hover y click
- Efectos 3D
- Código limpio y bien estructurado
